• <Notes> — General
    1. We can apply style to HTML elements by 2 ways style attribute | style element
    2. With the quotes gone and a space added, the editor thinks that property:value; is an attribute, just like style.
    3. ID must be unique on the page.
    4. Selectors are one part of the CSS ruleset. They indicate which HTML element(s) the rule is for.
    5. Declaration blocks are the other part of the CSS ruleset, they say how the rule will modify the elements indicated by selectors.
    6. CSS rules are composed of a selector followed by a declaration block.
    7. Type is the simplest kind of selector. It indicates the name of one type of HTML element (such as em or li).
    8. Class in HTML is an attribute that groups elements together that you want to have the same style.
    9. ID is an attribute that must be unique to a single element, used to identify it.
  • Type Selector
    1. Type Selector p { color: white }
    2. Class & ID Selector .follow-class { color: white } or ID #follow-id { color: white }
    3. Descendent (Combining) Selector li em { color: white }
  • DOM — Document Object Model It's a tree structure with a node for each HTML element, piece of text, image, and any other object in the web page.
